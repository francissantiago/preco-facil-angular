# Política de Privacidade - Preço Fácil

**Data de vigência:** 27 de fevereiro de 2026

Esta Política de Privacidade descreve como o aplicativo **Preço Fácil**, desenvolvido por Francis Santiago ("Desenvolvedor"), coleta, usa e protege as suas informações. Este aplicativo foi criado com foco na utilidade pública e privacidade, operando quase inteiramente de forma offline.

## 1. Coleta e Armazenamento de Dados Locais

O Preço Fácil **não** coleta, transmite ou armazena os seus dados pessoais, orçamentos, valores de hora técnica ou configurações em servidores externos.
Todos os dados inseridos no aplicativo (incluindo materiais e cálculos) são armazenados localmente no seu próprio dispositivo. Se você desinstalar o aplicativo ou limpar os dados de armazenamento do sistema, essas informações serão apagadas permanentemente.

## 2. Serviços de Terceiros e Monetização (Google AdMob)

Para manter o aplicativo gratuito e apoiar o desenvolvimento contínuo, utilizamos o serviço **Google AdMob** para a exibição de anúncios.
O Google AdMob atua como um fornecedor de terceiros e pode coletar e usar dados de forma automática para fornecer anúncios e métricas de desempenho. Os dados coletados pelo ecossistema do Google podem incluir:

- Endereço IP.
- Identificadores de publicidade do dispositivo (ID de publicidade do Android).
- Dados de interação e falhas de anúncios.

Para entender detalhadamente como o Google coleta e usa esses dados, consulte a [Política de Privacidade do Google](https://policies.google.com/privacy).

## 3. Permissões do Dispositivo

O aplicativo requer e utiliza as seguintes permissões básicas do sistema operacional para funcionar corretamente:

- **Acesso à Internet / Estado da Rede:** Necessário exclusivamente para solicitar e exibir os anúncios do Google AdMob.
- **Armazenamento em Cache:** Utilizado temporariamente para gerar e exportar os orçamentos em formato PDF para que você possa compartilhá-los via WhatsApp ou e-mail. Estes arquivos não são enviados para nós.

## 4. Transparência e Código Aberto

O Preço Fácil é um projeto de código aberto. O código-fonte está disponível publicamente no GitHub. Esta abordagem garante total transparência para a comunidade sobre o funcionamento do aplicativo, comprovando a ausência de coletores de dados ocultos.

## 5. Alterações nesta Política

Podemos atualizar a nossa Política de Privacidade de tempos em tempos. É recomendável revisar esta página periodicamente para verificar quaisquer alterações.

## 6. Contato

Se você tiver alguma dúvida, sugestão ou encontrar algum problema em relação a esta Política de Privacidade ou ao aplicativo, sinta-se à vontade para abrir uma _issue_ no repositório oficial do projeto no GitHub ou entrar em contato com o desenvolvedor, Francis Santiago.
